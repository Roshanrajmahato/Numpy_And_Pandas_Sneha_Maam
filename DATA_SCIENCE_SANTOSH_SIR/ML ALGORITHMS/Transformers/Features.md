# **Q15. What is a Feature?**

## **Interview-Ready Answer**

A **Feature** is an **individual measurable property or characteristic** of data that is used as an **input variable (X)** to help a machine learning model make predictions.

Features represent the **information** that the model uses to learn patterns and predict the **target variable (Y)**.

---

## **Simple Explanation**

A **feature** is simply **one column of input data** used to make predictions.

In short:

👉 **Features = Inputs (X)**
👉 **Target = Output (Y)**

---

## **Real-Time Example (Interview Friendly)**

Suppose you are building a **house price prediction model** 🏠.

### Features (Inputs X):

* **Size** → 2000 sqft
* **Bedrooms** → 3
* **Age of House** → 15 years
* **Location** → Downtown
* **Distance to Metro** → 0.5 km

### Target (Output Y):

* **House Price** → ₹45,00,000

Here:

👉 Each input (size, bedrooms, age, etc.) is a **feature**
👉 Price is the **target**

---

## **Types of Features**

### **1. Numerical Features**

Numbers that represent measurable quantities.

Examples:

* Age = 25
* Salary = ₹50,000
* Temperature = 32°C

Used in:

* Regression models
* Neural Networks

---

### **2. Categorical Features**

Features that represent **categories or labels**.

Examples:

* Gender = Male/Female
* City = Delhi/Mumbai/Bangalore
* Payment Type = Cash/Card/UPI

Usually converted using:

* **Label Encoding**
* **One-Hot Encoding**

---

### **3. Binary Features**

Features with only **two possible values**.

Examples:

* Yes/No
* True/False
* 0/1

---

## **Why Features are Important**

✔ Features determine **model performance**
✔ Good features → Better predictions
✔ Poor features → Poor accuracy
✔ Feature selection improves efficiency

---

## **Key Interview Tip ⚡**

If interviewer asks:

**"What is the difference between feature and target?"**

Say:

👉 *"Features are input variables used to make predictions, while the target is the output variable that the model tries to predict."*

That answer is **very commonly expected**.
